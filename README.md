# Food Delivery Admin Panel - Inspect Via Me Test

An admin panel testing and inspection project for a food delivery application.

## Overview

This repository contains a React-based admin panel interface for a food delivery application with integrated testing and inspection utilities. The admin panel allows restaurant managers and administrators to manage orders, restaurants, menus, and delivery tracking.

## Features

- User authentication and login
- Order management system
- Restaurant and menu management
- Delivery tracking and logistics
- User analytics and reporting
- Payment processing dashboard
- Responsive design for desktop and tablet

## Getting Started

### Prerequisites

- Git
- Node.js (v14 or higher)
- npm or yarn package manager
- React (v17 or higher)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nehanarang2533/Inspect-Via-Me-Test.git
cd Inspect-Via-Me-Test
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

1. Navigate to the login page (`login.html`)
2. Enter your credentials to access the admin panel
3. Manage orders, restaurants, and delivery operations
4. View analytics and reports
